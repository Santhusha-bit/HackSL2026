# Partner Logos

Logos are loaded from:

- **IEEE**: [Simple Icons CDN](https://cdn.simpleicons.org/ieee/0088CC) (reliable)
- **Others**: [Clearbit Logo API](https://logo.clearbit.com/) (e.g. `logo.clearbit.com/nibm.lk`)

If Clearbit fails to load, the component falls back to local files in this folder:

- `nibm.png` – NIBM
- `sliit.png` – SLIIT
- `uom.png` – University of Moratuwa
- `usj.png` – University of Sri Jayewardenepura
- `kdu.png` – Kotelawala Defence University
- `ieee.png` – IEEE

Add these PNG/JPG files here to use local logos instead.
